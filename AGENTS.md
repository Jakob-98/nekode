# AGENTS.md

macOS menubar app (CatAssistant) that monitors coding sessions from Claude Code, opencode, and arbitrary CLI commands. Shows session status in the menubar and as desktop pets — pixel-art animals that mirror each session's state. Includes `cathook` (shell hook for session tracking) and `catwait` (pipe-based CLI for monitoring any command).

Build: `make build` / `make install` / `make run`
